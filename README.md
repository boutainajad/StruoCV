==> Description
Application web moderne permettant de créer, personnaliser et télécharger des CV professionnels en quelques étapes simples. L'application offre une interface intuitive avec un formulaire à étapes, des validations en temps réel, et plusieurs modèles de CV personnalisables.
- Fonctionnalités
- Formulaire à étapes (Stepper)

Informations personnelles : Nom, photo, email, téléphone, adresse, LinkedIn, GitHub, portfolio
Détails professionnels : Titre de poste, résumé professionnel
Compétences : Hard skills et soft skills avec ajout dynamique
Langues : Liste des langues maîtrisées avec niveaux
Loisirs : Centres d'intérêt personnels
Formation : Cursus universitaire avec ajout dynamique
Expérience : Parcours professionnel détaillé
Certifications : Liste des certifications avec liens

==> Personnalisation

2 modèles de CV distincts et professionnels
Prévisualisation en temps réel de votre CV
Design responsive adapté à tous les écrans
Interface moderne avec Tailwind CSS

==> Validation et contrôle

Validation en temps réel des champs (email, téléphone, URL)
Messages d'erreur clairs pour guider l'utilisateur
Barre de progression pour suivre l'avancement
Navigation intuitive entre les étapes

==> Export et sauvegarde

Téléchargement PDF optimisé (< 1 Mo)
Option d'impression directe
Sauvegarde locale pour modifications futures
Export JSON des données du CV

==> Technologies utilisées

HTML5 : Structure sémantique
Tailwind CSS : Framework CSS utilitaire
JavaScript (Vanilla) : Logique applicative native
DOM API : Manipulation dynamique des éléments
LocalStorage : Sauvegarde des données
jsPDF : Génération de PDF (si utilisé)
